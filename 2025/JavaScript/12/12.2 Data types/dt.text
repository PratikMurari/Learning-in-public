### ðŸ§  1. **Predefined (Built-in) JavaScript Functions**

These are globally available functions you can call anytime:

| Function | Description |
|---------|-------------|
| `alert(message)` | Shows a popup with a message |
| `prompt(message)` | Asks the user for input with a popup textbox |
| `confirm(message)` | Asks the user to confirm (OK/Cancel) |
| `console.log(data)` | Prints output to the browser's console |
| `parseInt(string)` | Converts string to integer |
| `parseFloat(string)` | Converts string to float |
| `isNaN(value)` | Checks if a value is *Not-a-Number* |
| `Number(value)` | Converts to number |
| `String(value)` | Converts to string |
| `typeof value` | Returns the data type of a value |
| `eval(expression)` | Executes a string as JavaScript code (use with caution) |
| `setTimeout(fn, delay)` | Runs a function after a delay (ms) |
| `setInterval(fn, interval)` | Repeats a function every interval (ms) |
| `clearTimeout(id)` | Cancels a timeout |
| `clearInterval(id)` | Cancels a repeating interval |
