 ================================
📝 Manipulating Text with jQuery
=================================

👉 1. Change the text of an element:
   $("h1").text("Goodbye");       // Replaces text content of <h1> with "Goodbye"

👉 2. Change text for all matching elements:
   $("button").text("Click me!"); // Updates text for ALL <button> elements

👉 3. Add HTML inside an element:
   $("h1").html("<em>Hello</em>");  // Inserts HTML, will render as italic text

🧠 .text() vs .html():
- .text() → Inserts plain text only (ignores HTML tags)
- .html() → Inserts HTML content (interprets tags like <em>, <strong>, etc.)

================================
💡 Why jQuery Makes Life Easier
================================

✅ jQuery is a JavaScript library — written in JavaScript!
✅ It provides shortcut methods (e.g., .text(), .html(), .css(), etc.)
✅ Reduces code length and typing
✅ Makes DOM manipulation more intuitive

⚠️ But remember:
- jQuery is powerful because it's built on Vanilla JavaScript
- You should understand how Vanilla JS works first, then jQuery will make more sense
