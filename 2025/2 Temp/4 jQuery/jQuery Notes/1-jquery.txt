======================
🧠 jQuery Integration
======================

📥 HOW TO ADD JQUERY:
* Either:
   - Download the jQuery file,(https://jquery.com) or
   - (Recommended) Use a CDN (Content Delivery Network) like Google’s (https://developers.google.com/speed/libraries#jquery)

✅ Why Use a CDN?
- Faster loading if user already visited another site using the same CDN.
- Browser may cache jQuery → quicker load time.

💡 BASIC JQUERY SYNTAX:
- Example: Change <h1> color to red
    $("h1").css("color", "red");

❌ If jQuery Doesn't Work:
- Error: "$ is not defined" = jQuery not added yet

✅ ADDING jQuery:
- Use CDN script ABOVE your own Js script:
  <script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
  <script src="index.js"></script>

⚠️ SCRIPT ORDER MATTERS:
- Browser reads code top-to-bottom.
- If index.js loads before jQuery, $ won’t be recognized.

💡 BEST PRACTICES:
1. Put all <script> tags at end of <body>
   → Ensures HTML loads before JS runs

OR

2. Use this pattern to wait for DOM to load:
   $(document).ready(function() {
     // jQuery code here
   });

📝 Summary:
- Always load jQuery before your script
- Use $(document).ready() if scripts are in <head>
- Or better: keep scripts at end of <body>
