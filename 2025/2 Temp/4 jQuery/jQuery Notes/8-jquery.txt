======================
🎬 jQuery Animations
======================

👉 1. Hide, Show, Toggle:
   $("h1").hide();           // Instantly hides the element
   $("h1").show();           // Shows the hidden element
   $("h1").toggle();         // Toggles between show and hide

👉 2. Fade Animations:
   $("h1").fadeOut();        // Gradually fades out (reduces opacity then hides)
   $("h1").fadeIn();         // Fades back in
   $("h1").fadeToggle();     // Toggles between fadeIn and fadeOut

👉 3. Slide Animations:
   $("h1").slideUp();        // Slides element up (collapses it)
   $("h1").slideDown();      // Slides it back down (reveals)
   $("h1").slideToggle();    // Toggles slideUp and slideDown

👉 4. Custom Animations with .animate():
   $("h1").animate({ opacity: 0.5 });               // Changes opacity to 50%
   $("h1").animate({ margin: "20px" });             // Changes margin
   $("h1").animate({ width: "50%" });               // Changes width

⚠️ Note:
- Only CSS properties with numeric values (like `margin`, `width`, `opacity`, etc.) can be used with `.animate()`.
- You cannot animate color using `.animate()`.

👉 5. Chaining Animations:
   $("h1").slideUp().slideDown().animate({ opacity: 0.5 });

✅ Animations run **in sequence** when chained.
✅ Great for creating interactive UI effects (e.g., dropdowns, loaders, transitions)
